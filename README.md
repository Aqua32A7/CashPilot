# CashPilot 🚀

CashPilot is a modern **startup runway calculator** designed to help founders, startups, and small businesses estimate how long their available cash will last based on monthly burn rate and revenue.

The project was built using **HTML, CSS, and JavaScript** with a focus on creating a premium user experience, responsive design, and practical financial insights.

---

## 🌟 Features

* 💰 Enter Total Cash on Hand
* 📉 Enter Monthly Burn Rate
* 📈 Optional Monthly Revenue Input
* ⚡ Instant Runway Calculation
* 🟢 Safe / 🟡 Warning / 🔴 Danger Status Zones
* 📅 Estimated Zero Cash Date
* 🎯 Pre-built Startup Scenarios
* 📱 Fully Responsive Design
* ✨ Smooth Animations & Modern UI

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

## 📌 How It Works

CashPilot uses the following formula:

Runway = Total Cash on Hand ÷ Net Monthly Burn

Where:

Net Monthly Burn = Monthly Expenses - Monthly Revenue

The tool also supports scenario planning and dynamic financial forecasting.

---

## 🎯 Why I Built This

Most runway calculators are very basic and lack modern design.
CashPilot was created to combine useful financial planning tools with a polished SaaS-style user experience.
